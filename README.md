# Daily Dog Planner

Mit dem [Daily Dog Planner](https://capstone-project-nine-gamma.vercel.app/) bist du als Hundebesitzer immer bestens über deinen Hund informiert.

Lege ein Profil an und verwalte tierärztliche Untersuchungen sowie wichtige Termine. Ideen, Neuanschaffungen oder Allergien lassen sich einfach in den Notizen festhalten.

> Diese App ist für die mobile Nutzung optimiert.

## Lokale Installation

```bash
npm install
npm run dev
```

Anschließend die App unter [http://localhost:5173](http://localhost:5173) im Browser öffnen.

## Tech Stack

- React 19
- React Router v7
- Vite 8
- Framer Motion
- Styled Components
- PWA (vite-plugin-pwa)
